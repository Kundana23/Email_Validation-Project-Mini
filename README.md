email_validate.py is the python code written for email validation testing.
This code is in both Interactive mode and Batch mode

input.txt is the dataset with combination of valid and invalid email ids

out.txt is the empty file for batch mode to write the results after running the code and taking input.txt in read mode

result_after_validation_output.txt is the batch mode result of the program that gets displayed in out.txt after execution and also contain the output presented in console
